eacher Intelligence System (TIS)

Version: 0.1
Status: Entwurf

Projektziel

Entwicklung eines persönlichen, KI-gestützten Systems zur Verwaltung eigener Lerngruppen.

Das System soll Lehrkräfte bei Dokumentation, Leistungsbewertung, Lernentwicklung und Berichtserstellung unterstützen.

Das System ist zunächst für einzelne Lehrkräfte konzipiert und nicht als vollständige Schulverwaltungssoftware.

Benutzerrollen
Rolle: Lehrkraft

Die Lehrkraft kann:

Schüler verwalten
Klassen verwalten
Fehlzeiten erfassen
Kompetenzen bewerten
Noten verwalten
Dokumente hochladen
Beobachtungen dokumentieren
KI-Abfragen durchführen
Berichte erzeugen
Muss-Anforderungen
REQ-001 Schülerverwaltung

Schüler können einzeln oder per Import angelegt werden.

REQ-002 Klassenverwaltung

Klassen und Lerngruppen können angelegt werden.

REQ-003 Fehlzeiten

Fehlzeiten können erfasst und ausgewertet werden.

REQ-004 WebUntis-Import

Fehlzeiten sollen aus WebUntis übernommen werden können.

REQ-005 Kompetenzraster

Kompetenzen können frei definiert werden.

REQ-006 Kompetenzbewertung

Bewertungsskalen müssen konfigurierbar sein.

Beispiel:

Trifft nicht zu
Trifft teilweise zu
Trifft fast zu
Trifft zu
REQ-007 Notenverwaltung

Noten und Notenpunkte können gespeichert werden.

REQ-008 Schülerakte

Jeder Schüler besitzt eine digitale Akte.

REQ-009 Dokumentenablage

Dateien können Schülern zugeordnet werden.

REQ-010 Beobachtungen

Unterrichtsbeobachtungen können gespeichert werden.

REQ-011 KI-Abfragen

Die KI kann Fragen zu Schülerdaten beantworten.

REQ-012 Lernentwicklungsberichte

Die KI kann Lernentwicklungsberichte erzeugen.
